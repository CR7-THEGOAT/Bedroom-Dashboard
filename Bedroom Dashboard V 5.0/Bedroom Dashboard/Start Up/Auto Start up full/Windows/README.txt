Bedroom Dashboard auto-start for Windows

Run this once to start Bedroom Dashboard automatically after login and recover it after sleep.

Run:
powershell -ExecutionPolicy Bypass -File ".\INSTALL-AUTO-START-WINDOWS.ps1"

It installs:
- Bedroom Dashboard Autostart
- Bedroom Dashboard Wake Recover
- Bedroom Dashboard Watchdog

Logs:
C:\Users\saeed\OneDrive\Documents\Bedroom Dashboard\logs\auto-restart.log
C:\Users\saeed\OneDrive\Documents\Bedroom Dashboard\logs\watchdog.log
