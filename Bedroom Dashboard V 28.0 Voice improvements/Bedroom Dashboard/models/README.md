# KISOKE Local Models

KISOKE is prepared for Ollama running locally at `http://localhost:11434`.

Recommended models:

```bash
ollama pull llama3.2:1b
ollama pull qwen2.5:3b
ollama pull qwen3:4b
```

Easy model: `qwen2.5:3b`

Hard model: `qwen3:4b`

Backup small model: `llama3.2:1b`

The frontend command router works without Ollama. Ollama is only needed for open-ended AI questions.
