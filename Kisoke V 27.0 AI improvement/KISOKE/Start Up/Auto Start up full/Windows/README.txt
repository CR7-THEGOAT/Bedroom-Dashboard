KISOKE auto-start for Windows

Run this once to start KISOKE automatically after login and recover it after sleep.

Run:
powershell -ExecutionPolicy Bypass -File ".\INSTALL-AUTO-START-WINDOWS.ps1"

It installs:
- KISOKE Autostart
- KISOKE Wake Recover
- KISOKE Watchdog

Logs:
C:\Users\saeed\OneDrive\Documents\KISOKE\logs\auto-restart.log
C:\Users\saeed\OneDrive\Documents\KISOKE\logs\watchdog.log
