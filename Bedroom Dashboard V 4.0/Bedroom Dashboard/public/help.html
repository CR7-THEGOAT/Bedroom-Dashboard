Project Nexora

Help Sheet

Start

Open a terminal in the Bedroom Dashboard folder and run:

npm install
npm run dev

Open http://localhost:5173 on the kiosk laptop.

Other Devices

Keep both devices on the same Wi-Fi. In Settings, use the Help / Other Devices card and open the shown IP link from the other device.

http://192.168.1.25:5173

Daily Mode

For a more stable daily setup:

npm run build
npm start

Open http://localhost:4173.

Firewall

If another device cannot open it, allow the port on Linux:

sudo ufw allow 5173/tcp
sudo ufw allow 4173/tcp

Fullscreen Kiosk

chromium --kiosk http://localhost:5173

Use http://localhost:4173 instead if you started with npm start.
